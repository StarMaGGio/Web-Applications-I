# Web Applications I 🌐

This repository contains the evolution of the **Film Library** web application developed during the labs of the course **Web Applications I** of **Politecnico di Torino**.

## 📁 Repository structure
* `Lab01`: basic javascript application.
* `Lab02`: database implementation with sqlite3 and Promises.
* `Lab03`: API development with express.
* `Lab04`: front-end design with HTML, CSS and Bootstrap.
